# CodeAlpha_AIChatbot
AIChatbot
Simple Java AI Chatbot
A simple, interactive console-based AI Chatbot built in Java. It responds to standard greetings, questions, and shuts down gracefully upon command.

Features
Interactive Console Input: Uses standard input to communicate with the user.
Greeting Responses: Recognizes greetings such as "hello" and "hi".
Basic Info Queries: Responds to queries about its "name" and "how are you".
Limitations Acknowledgment: Explicitly handles unsupported queries like "time".
Graceful Termination: Shuts down when the user types "bye".
Getting Started
Prerequisites
Java Development Kit (JDK) installed (version 8 or higher is recommended).
Compilation
To compile the chatbot, open your terminal or command prompt, navigate to the project directory, and run:

javac AIChatbot.java
Running the Application
After compilation, run the chatbot using:

java AIChatbot
Example Interaction
AI Chatbot Started! Type 'bye' to exit.
You: hi
Bot: Hello! How can I help you?
You: what is your name?
Bot: I am a Java AI Chatbot.
You: how are you
Bot: I am fine. Thank you!
You: bye
Bot: Goodbye! Have a nice day.
